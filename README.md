# react-native-model-viewer

GLB and GLTF model viewer

## Installation

```sh
npm install react-native-model-viewer
```

## Usage

```js
import { ModelViewerView } from "react-native-model-viewer";

// ...

<ModelViewerView color="tomato" />
```
